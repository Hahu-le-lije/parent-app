Story Game
track
StorySession {
  story_id
  pages_read: number
  total_pages: number

  keywords_clicked: number
  unique_keywords_clicked: string[]

  time_spent: number

  questions: [
    {
      question_id,
      is_correct,
      attempts
    }
  ]

  
}
Scoring Logic
reading_score = pages_read / total_pages * 30
quiz_score = correct_answers / total_questions * 50
engagement_score = min(keywords_clicked / total_keywords, 1) * 20

final_score = reading_score + quiz_score + engagement_score

Skill Breakdown
{
  reading_completion: 0–1,
  comprehension: 0–1,
  vocabulary_interest: 0–1
}
Insights
Child skips pages → low attention
High keyword clicks → curious learner
Low quiz + high reading → decoding but not understandingPicture → Word Game

Word Builder
Track
{
  words_found: []
  total_possible_words
  wrong_attempts
  hints_used
  time_taken
}
Score
completion = words_found / total_words * 50
efficiency = (1 - wrong_attempts_ratio) * 20
independence = (1 - hints_used_ratio) * 20
speed = time_score * 10

final_score = completion + efficiency + independence + speed
Skills
{
  spelling,
  pattern_recognition,
  vocabulary,
  problem_solving
}
Insights
Uses many hints → needs support
Finds rare words → advanced learner
